    C дёҺ C++

    • C++ и®Өдёә int fun()==int fun(void), C и®ӨдёәеҸӘжҳҜзңҒз•ҘдәҶ
    • C зҡ„ char еёёйҮҸ 'a' жҳҜ int зұ»еһӢ, еӣ жӯӨ 'ABCD' дёҚдјҡжҠҘй”ҷ
    • C++зҡ„ const еҸҳйҮҸй»ҳи®ӨжҳҜеҶ…йғЁй“ҫжҺҘ, еҸҜд»Ҙз”Ё extern ж”№дёәеӨ–йғЁй“ҫжҺҘ
    • C++ зҡ„ const еҸҳйҮҸеҸҜд»ҘеҲқе§ӢеҢ– const, C дёҚиЎҢ
    • C++ дёҘж јдҪҝз”Ёжһҡдёҫ, C е°Ҷд№Ӣи®Өе®ҡдёәж•ҙеһӢ
    • C++ еҺҹз”ҹе®Ҫеӯ—з¬Ұ, C еҺҹз”ҹеӨҚж•°
    • C++11 жІЎжңүеҲқе§ӢеҢ–еҷЁ, restrict, VLA, дјёзј©еһӢж•°з»„жҲҗе‘ҳ, еҸҜеҸҳж•°йҮҸеҸӮж•°зҡ„е®Ҹ
    • C++ ж Үи®°еҗҚ (иҮӘе®ҡд№үзұ»еһӢеҗҚжҲ–ж ҮзӯҫеҗҚ) дёҺеҸҳйҮҸеҗҚеҸҜд»ҘйҮҚеӨҚ, C дёҚиЎҢ
    • molloc еңЁ C++ дёӯеҝ…йЎ»ејәеҲ¶зұ»еһӢиҪ¬жҚў
    • C иӯҰе‘Ҡе°Ҷ const жҢҮй’ҲиөӢз»ҷжҷ®йҖҡжҢҮй’Ҳ, C++ error
    • C++ еҸҜд»Ҙз”Ё const ж•ҙеһӢеЈ°жҳҺж•°з»„еӨ§е°Ҹ
    • C++ еҸҜд»Ҙ type(name) ејәеҲ¶зұ»еһӢиҪ¬жҚў
    • C++ дёҚеҸҜи°ғз”Ё main() зҺ°еңЁ C д№ҹдёҚиЎҢдәҶ

    д»ҘдёҠеҸ—ж—¶й—ҙеҪұе“Қ, д»…дҫӣеҸӮиҖғ

    еЈ°жҳҺ

    • жһ„жҲҗ: type name+ дҝ®йҘ°з¬Ұ
    • дёҚи®Ёи®ә const extern зӯү
    • дҝ®йҘ°з¬Ұжңүдёүз§Қ:
      • еҗҺзҪ®жңү [] дёҺ (), дјҳе…Ҳзә§зӣёеҗҢ, иҮӘе·Ұеҗ‘еҸіз»“еҗҲ
      • еҪ“з„¶, иҝҷжҳҜеҜ№и§„еҫӢзҡ„жҖ»з»“, е®һйҷ…дёҠе®ғд»¬дёҚжҳҜиҝҗз®—з¬Ұ, жІЎжңүдјҳе…Ҳзә§дёҺз»“еҗҲеҫӢ
      • еүҚзҪ® * дјҳе…Ҳзә§е°ҸдәҺеҗҺзҪ®, еӣ жӯӨжҲ‘д»¬еЈ°жҳҺж•°з»„дёҺеҮҪж•°зҡ„жҢҮй’Ҳеҝ…йЎ»дҪҝз”Ё ()
      • е°Ҷдҝ®йҘ°з¬ҰдёҺ name з»“еҗҲ, зҗҶи§Је®ғзҡ„зұ»еһӢ
      • жңҖеҗҺдҪ дјҡеҸ‘зҺ°, еҰӮжһңдҪ е°Ҷд»ҘдёҠдёүз§Қз¬ҰеҸ·и®ӨдёәжҳҜиҝҗз®—з¬ҰеҪўејҸ, йӮЈд№ҲдҪ еҫ—еҲ°зҡ„жҳҜдёҖдёӘиЎЁиҫҫејҸ, иҖҢеүҚйқўзҡ„ type е°ұжҳҜиЎЁиҫҫејҸеҖјзҡ„зұ»еһӢ

    й“ҫжҺҘж•°з»„

    • еңЁй“ҫжҺҘйҳ¶ж®өи·Ёж–Үд»¶дҪҝз”ЁдёҖдёӘе…ЁеұҖж•°з»„зҡ„ж—¶еҖҷ
    • дёҚиғҪ extern дёҖдёӘжҢҮй’Ҳ
    • еӣ дёәй“ҫжҺҘеҸҳйҮҸжҳҜжҢҮеҮә a зҡ„ең°еқҖ (з”ұеҸҰдёҖдёӘж–Үд»¶е®ҡд№ү), еҪ“дҪ жҠҠдёҖдёӘж•°з»„еЈ°жҳҺдёәеҜ№еә”е…ғзҙ зҡ„жҢҮй’Ҳ, дјҡи®Өдёәж•°з»„ең°еқҖеӯҳеӮЁзҡ„йӮЈдёӘеҖјжҳҜжҢҮй’Ҳзҡ„еҖј, жң¬иҙЁеҺҹеӣ жҳҜж•°з»„зҡ„еҖјиҷҪз„¶жҳҜйҰ–е…ғзҙ зҡ„жҢҮй’Ҳ, дҪҶжІЎжңүдҪҝз”ЁйўқеӨ–зҡ„з©әй—ҙеӯҳеӮЁ, зЁӢеәҸдјҡи®ҝй—®жңӘзҹҘзҡ„еҶ…еӯҳ

    еҜ№йҪҗ

    • еҶ…еӯҳеҜ№йҪҗжҳҜжҢҮж•°жҚ®еңЁеҶ…еӯҳдёӯеӯҳеӮЁзҡ„иө·е§Ӣең°еқҖеҝ…йЎ»жҳҜжҹҗдёӘзү№е®ҡеҖјзҡ„ж•ҙж•°еҖҚ
    • еҶ…еӯҳеҜ№йҪҗзҡ„еҘҪеӨ„
      • CPU е’ҢеҶ…еӯҳд№Ӣй—ҙзҡ„ж•°жҚ®жҖ»зәҝжҳҜжңүе®ҪеәҰзҡ„, дҫӢеҰӮ 64 дҪҚ CPU, ж•°жҚ®жҖ»зәҝйҖҡеёёжҳҜ 64 дҪҚпјҲ8 еӯ—иҠӮпјү
      • еҶ…еӯҳеӯҗзі»з»ҹд№ҹжҳҜжҢүз…§еқ—жҲ–еӯ—жқҘеӯҳеҸ–зҡ„
      • x86/x64 иҰҒжұӮж•°жҚ®еҜ№йҪҗ, еҗҰеҲҷдјҡеҜјиҮҙжҖ§иғҪдёӢйҷҚ (еӨҡж¬ЎеҸ–, дҪҚз§»еҗҲе№¶), еҸҰеӨ–и®ёеӨҡжһ¶жһ„е№Іи„ҶдёҚж”ҜжҢҒжңӘеҜ№йҪҗи®ҝй—®
      • еҸҰеӨ–еҶҷе…ҘдёҖе®ҡжҳҜеҜ№йҪҗзҡ„, иҜ»еҰӮжһңдёҚеҜ№йҪҗ, дјҡж’•иЈӮ, еҜ№йҪҗиҝҳдјҡз®ҖеҢ–еҰӮ SIMD зӯүжҢҮд»Өзҡ„е®һзҺ°
    • C дёҺ C++ зҡ„з»“жһ„дҪ“еҜ№йҪҗеӨ„зҗҶ (жҸ’е…Ҙж— ж„Ҹд№үзҡ„еӯ—иҠӮ (Padding), еҺҹз”ҹзұ»еһӢиӮҜе®ҡдёҚеҝ…з®Ў)
      • жҜҸдёӘжҲҗе‘ҳзҡ„еҒҸз§»йҮҸеҝ…йЎ»жҳҜиҜҘжҲҗе‘ҳиҮӘиә«еҜ№йҪҗеҖјзҡ„еҖҚж•°
      • ж•ҙдёӘз»“жһ„дҪ“зҡ„еӨ§е°Ҹеҝ…йЎ»жҳҜиҜҘз»“жһ„дҪ“дёӯжңҖеӨ§жҲҗе‘ҳеҜ№йҪҗеҖјзҡ„еҖҚж•° (дёәдәҶж”ҜжҢҒз»“жһ„дҪ“ж•°з»„иҝҷдёҖеңәжҷҜ, дҝқиҜҒж•°з»„з¬¬дәҢдёӘе…ғзҙ д№ҹжҳҜеҜ№йҪҗзҡ„)
    • зү№ж®Ҡжғ…еҶө
      • з©әз»“жһ„дҪ“зҡ„еӨ§е°Ҹ
        • C иҜӯиЁҖдёӯжңӘе®ҡд№ү
        • C++ и§„е®ҡиҮіе°‘дёә 1
      • иҷҡеҮҪж•°дёҺз»§жүҝзҡ„еҪұе“Қ
        • vptr дјҡж”№еҸҳзұ»зҡ„еёғеұҖе’ҢеҜ№йҪҗиҰҒжұӮ (жҢүе№іеҸ°жҢҮй’ҲеӨ§е°Ҹ)
        • жҙҫз”ҹзұ»дјҡеӨҚз”Ёеҹәзұ»зҡ„еЎ«е……еӯ—иҠӮ
    • еҜ№йҪҗж“Қзәө
      • alignas(N), еҸҜд»ҘжҢҮе®ҡеҜ№йҪҗеҖј
      • alignof(T), еҸҜд»ҘиҺ·еҸ–зұ»еһӢ T зҡ„еҜ№йҪҗеҖј
      • #pragma pack(N), еҸҜд»ҘжҢҮе®ҡеҜ№йҪҗеҖј, иҢғеӣҙ 1-16, й»ҳи®Өдёә 8
        • дёҖе®ҡиҰҒе°ҸеҝғдҪҝз”Ё pointers жҢҮеҗ‘ packed struct зҡ„жҲҗе‘ҳ
      • жңүж—¶жҲ‘д»¬еҲ©з”Ё alignas(N) ејәеҲ¶иҝҮеәҰеҜ№йҪҗ
        • еҸҜд»ҘйҒҝе…ҚдјӘе…ұдә« (еӨҡзәҝзЁӢдёӯ, дёӨдёӘзәҝзЁӢйў‘з№Ғдҝ®ж”№зҡ„еҸҳйҮҸеҰӮжһңеӨ„дәҺеҗҢдёҖдёӘзј“еӯҳиЎҢ, дјҡеҜјиҮҙзј“еӯҳдёҖиҮҙжҖ§еҚҸи®®йў‘з№ҒеңЁж ёеҝғй—ҙзј“еӯҳиЎҢ, йҖ жҲҗжҠ–еҠЁ)
    • ж ҮеҮҶзҡ„ malloc еңЁ 64 дҪҚзі»з»ҹдёҠйҖҡеёёдҝқиҜҒ 16 еӯ—иҠӮеҜ№йҪҗ (дёәдәҶйҖӮй…Қ long double жҲ– SSE жҢҮд»Ө (SIMD жӢ“еұ•) зҡ„еҹәзЎҖиҰҒжұӮ)
    • еҰӮжһңдҪ йңҖиҰҒ 64 еӯ—иҠӮеҜ№йҪҗ(дёәдәҶ AVX-512), ж ҮеҮҶ new еҸҜиғҪеҒҡдёҚеҲ°
      • йңҖиҰҒдҪҝз”Ё _mm_malloc (Windows/Intel) жҲ– posix_memalign (Linux), жҲ–иҖ… C++17 зҡ„ std::aligned_alloc

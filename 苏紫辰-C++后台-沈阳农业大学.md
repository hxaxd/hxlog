## 项目经历

### 轻量级分布式 KV 存储系统

**项目描述**：

为深入理解分布式存储系统的核心机制，**从零设计并实现一个支持高可用、持久化与水平扩展的内存型 KV 存储系统**，对标 Redis Cluster 的基础能力，聚焦数据分片、复制容灾与性能优化。  

**主要工作**：

- **设计并实现基于跳表（SkipList）的内存存储引擎**，相比哈希表扩展了有序遍历能力，适用于带 TTL 和区间查询的场景；  
- **采用一致性哈希 + 虚拟节点实现数据分片**，显著降低节点扩缩容时的数据迁移比例，提升集群弹性与运维效率；  
- **实现主从异步复制机制**，通过自研 binlog 格式保证操作顺序性，结合心跳检测与超时选举策略，实现**秒级自动故障转移**；  
- **设计 RDB 快照 + AOF 日志混合持久化方案**，兼顾快速恢复与数据安全性，系统崩溃后**数据丢失窗口控制在亚秒级**；  
- **引入读写锁与对象内存池**，有效减少锁竞争与频繁堆分配开销，**显著提升多线程写入吞吐能力**。

### 轻量级 RPC 框架

**项目描述**：

**设计并实现一个支持服务发现、负载均衡、容错治理与跨语言调用的轻量级 RPC 框架**，聚焦高性能网络模型与服务治理能力。  

**主要工作**：

- **基于 epoll 实现主从 Reactor 网络模型**，单线程可支撑**万级并发连接**，避免传统多线程模型的上下文切换与资源竞争开销；  
- **设计轻量二进制协议头 + Protobuf 载荷**，在保证跨语言兼容性的同时，**有效降低序列化体积与解析开销**；  
- **基于 ZooKeeper 实现服务注册中心**，支持服务节点动态上线/下线、故障自动摘除与健康检测，保障服务拓扑实时一致性；
- **设计双模式负载均衡**，集成加权轮询（适应服务节点性能差异）与一致性哈希（适配缓存/状态敏感场景）；
- **基于 Future/Promise 实现异步调用接口**，支持批量请求合并，**显著减少高频调用场景下的网络往返次数**；  
- **引入熔断器（基于滑动窗口错误率统计）与超时重试机制**，在依赖服务异常时自动触发降级，保障调用链整体稳定性；

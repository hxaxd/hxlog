# 调试与可观测性

## 领域边界

- 调试用于定位已经观察到的错误
- Profiling 用于定位程序的时间, 内存与系统资源瓶颈
- 可观测性通过日志, Metrics 和 Tracing 理解运行中系统的内部状态

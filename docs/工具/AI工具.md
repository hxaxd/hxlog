## Claude code

### Claude code 命令

- `/init` 初始化项目, 收集项目信息
- `/compact text` 压缩上下文, text 作为指导
- `/clear` 清除上下文
- `think / think hard / think harder / urtrathink text` 控制思考长度
- `! command` 执行终端命令
- `# text` 作为记忆
- `/ide` 联动 IDE
- `ccr code -p text` 问问题
- `claude mcp add name --scope user --command 参数` 添加 mcp
- `.claude/command` 文件夹中的 md 文件为自定义命令
- `/agent` 创建智能体
- `resume` 历史话题

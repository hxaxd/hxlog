# 远程开发与集群

## 领域边界

- 在远程主机或集群上安全, 稳定地开发和运行任务
- 管理代码, 数据, 环境, 任务和运行结果在本地与远程之间的流动
- 这里只覆盖开发者使用集群所需的工具, 不展开分布式系统理论

## 谓词逻辑

- 个体词 / 谓词 / 量词
- 前束范式 所有的量词都在最左边且辖域覆盖全式
  - $Q_1x_1 Q_2x_2 \cdots Q_nx_n \ \varphi$
  - 其中 $Q_i \in { \forall, \exists }$, $\varphi$ 为不含量词的公式
- 解释 使全式为真的一组真值
- 推理规则

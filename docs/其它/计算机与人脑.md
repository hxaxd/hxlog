# 计算机与人脑

## 参考资料

- 计算机与人脑

## 计算机

- 分析了模拟计算机与数字计算机, 指出数字计算机更加强大

## 人脑

- 大脑是一种独特的混合体
    - 底层机制包含模拟过程 (电化学信号及其复杂, 还有浓度差别)
    - 输出结果表现为数字信号 (动作电位有或无)
    - 这种数字信号的频率也显示为模拟的
- 人脑计算其实很慢, 但是能耗比极优
- 但是人脑为何如此强大, 并行性

## 人脑之谜

- 为何高智能
    - 大脑的逻辑深度一定很浅 (防止误差累积)
    - 精度很低
    - 全靠并行性
- 为何高容错
    - 信息不在单一节点
    - 信息的传递依赖多个同含义路径的投票
- 大脑的计算模型
    - 一定不是数学
    - 接近于统计学
    - 处理相关性, 概率, 加权平均
    - 他怀疑无法用数学对大脑建模
- 大脑的存储模型
    - 远超计算机的存储能力
    - 存算不分离

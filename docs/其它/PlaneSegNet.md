# PlaneSegNet: A deep learning network with plane attention for plant point cloud segmentation in agricultural environments

## 参考资料

- [PlaneSegNet: A deep learning network with plane attention for plant point cloud segmentation in agricultural environments](https://www.sciencedirect.com/science/article/pii/S2589721725000947)

## 1. 引言

1. 植物表型测定是解析基因型-环境互作关系的核心，三维LiDAR技术是当前高通量表型测定的核心技术，但真实农业场景的群体点云包含大量非植物点，会严重降低株高、冠层覆盖度等下游表型性状估算的精度，在高密度植被、非平整地形场景中，非植物点的精准剔除仍是行业核心瓶颈。
2. 现有传统方法的局限：以DTM、高度阈值法、RANSAC、CSF、Grid Ground、PTD为代表的传统地面分割方法，仅能在特定条件下取得较好效果，在复杂地形适配、密集植被下精准分割、非植物结构识别、计算效率与泛化性平衡方面存在明显局限，且对参数设置高度敏感，需要大量人工调参。
3. 现有深度学习方法的局限：PointNet系列、PointCNN、RandLA-Net等深度学习模型虽在植物分割任务中取得了进步，但基于点的模型对点云采样策略高度依赖，稀疏卷积网络在植物与非植物边界模糊的农业场景中仍存在明显短板，行业内缺乏专门针对农业场景、优化边界分割性能的方案。
4. 本文研究思路：受人工标注中利用多视图投影区分植物结构与地面的思路启发，设计了平面注意力机制，通过XZ与YZ平面的投影引导融合增强空间特征，构建PlaneSegNet网络，专门提升植物与非植物边界区域的分割精度，适配多尺度农业场景。

## 2. 数据集与方法

### 2.1 数据集

核心内容：本研究采用了大田群体、温室栽培环境、大规模乡村景观（DALES）三类不同空间尺度的农业群体点云数据集，分别用于评估模型对复杂农业场景的适应性、可控环境下的跨物种泛化能力、高噪声大尺度场景下的鲁棒性，从多个维度全面验证模型的综合性能。

#### 2.1.1 大田群体点云数据

核心内容：

1. 数据集基础：采集了沈阳农业大学试验农场4种种植密度、4个关键生育期的玉米群体点云，通过FARO地面激光扫描仪完成数据采集，后续做了体素下采样处理，并通过人工标注将非植物点标为0类、植物点标为1类。
2. 数据划分策略：16组玉米点云垂直均分为两部分，上半部分用于训练、下半部分专门用于测试；训练集随机裁剪4m×4m的子区域，选取200个样本按9:1的比例划分为训练集与验证集；另外4组外部公开玉米数据集不参与训练，仅用于测试模型的跨数据源、跨品种泛化能力。
3. 数据增强：采用随机丢弃、随机旋转、随机缩放、随机翻转、随机抖动、弹性畸变、球形裁剪7种数据增强策略，丰富训练数据的多样性。
4. 数据集规模：总计使用20组玉米群体点云样本，覆盖了不同种植密度、生育期、数据源、采集方式，为模型的全面评估提供了坚实基础。

#### 2.1.2 温室群体点云数据

核心内容：

1. 数据集来源：采用公开的Crop3D温室数据集，包含玉米、油菜、马铃薯三类盆栽作物，同时还有机械结构、地面杂草等复杂背景元素，是验证模型在复杂背景下植物提取鲁棒性的理想测试集。
2. 数据划分与实验目的：沿用原数据集的划分方式，35个样本用于训练、15个样本用于测试，测试集覆盖3种作物类型，用于全面评估PlaneSegNet在温室环境多物种植物分割任务中的适应性与泛化能力。

#### 2.1.3 乡村机载激光雷达点云数据

核心内容：

1. 数据集来源：采用DALES公开基准数据集，由美国俄亥俄州代顿地区的机载激光扫描仪采集，覆盖范围约10km²，包含超5.05亿个点，每个点被标注为8个语义类别，是大规模户外场景的权威基准数据集。
2. 数据集特点与实验目的：相比大田和温室数据集，该数据集空间覆盖更广、地形起伏更大、背景噪声水平更高，用于严苛检验PlaneSegNet在大规模点云分割场景中的稳定性与鲁棒性。
3. 补充分析：三类数据集中非植物点均占据数据主体，且分布不规则、空间结构复杂，人工剔除成本极高，凸显了自动化精准植物点云分割的必要性。

### 2.2 方法

核心内容：针对农业场景中植物点与背景噪声难区分的核心挑战，提出PlaneSegNet分割模型，该模型以编码器-解码器（UNet）架构为基础，融入平面注意力模块提升边界分割精度，可实现群体级点云空间分布模式的端到端学习，能适配不同生育期、种植密度、复杂背景的农业场景。

#### 2.2.1 网络架构

核心内容：

1. 输入预处理：将农业点云转换为带三维坐标与法向量的格式，通过体素化把非结构化点云转为规则的稀疏体素栅格，在降低计算量的同时保留原始点云的局部几何结构与拓扑关系，同时保留索引映射，确保分割结果能反向投影回原始点云。
2. 编码器模块：先通过核尺寸5×5×5的子流形稀疏卷积层提取局部几何特征，再由多个下采样层+平面块组成；下采样采用稀疏卷积，逐步将空间分辨率减半、提升特征维度，每个下采样操作后均接入平面块，实现深层几何特征的增强提取。
3. 解码器模块：由多个上采样层+平面块组成，上采样通过稀疏逆卷积恢复空间分辨率，同时融合编码器中相同空间分辨率的多尺度特征，特征拼接后送入平面块做进一步增强，优化几何特征模糊区域的分割效果。
4. 输出处理：通过1×1×1的子流形稀疏卷积层生成体素级的语义类别概率分布，区分植物与非植物两类，再利用体素化过程中建立的索引映射，将体素级分割结果反向投影到原始点云，得到点级的语义分割结果，形成完整的大规模点云分割解决方案。

#### 2.2.2 平面注意力

核心内容：

1. 设计初衷：解决点云固有的稀疏性、三维分布不规则导致的细粒度几何结构难捕捉的问题，受人工标注用XZ/YZ投影区分植物与背景的思路启发，将三维特征与二维平面投影相融合，增强模型的空间几何表达能力。
2. 核心操作流程：先通过子流形稀疏卷积提取三维体素的局部几何特征；再将体素栅格投影到XZ、YZ两个垂直平面，构建平面注意力图，通过均值池化将三维特征聚合到对应的二维平面位置；随后将二维投影特征对齐恢复到三维体素空间，与原始三维特征拼接融合；最后通过两层MLP完成特征降维，加入残差连接输出增强后的特征。
3. 核心优势：能同时捕捉点云的局部与全局几何特征，强化垂直平面的上下文依赖关系，在农业环境中作物与非植物元素的边界区域分割任务中，具备显著的性能优势。

#### 2.2.3 损失函数

核心内容：

1. 损失函数选择：采用交叉熵损失作为模型的核心优化目标，用于衡量预测分布与真实标签分布之间的差异，可同时适配植物/非植物二分类任务，以及DALES数据集的8类别多分类任务。
2. 训练配置：所有模型训练均在搭载NVIDIA RTX A6000 GPU与128GB内存的工作站完成，优化器采用AdamW，初始学习率设置为0.006，搭配融合余弦退火策略的OneCycleLR学习率调度器，模型训练轮次为100轮，确保参数优化到最优状态。

#### 2.2.4 评价指标

核心内容：

1. 核心定量指标：采用交并比（IoU）与F1分数作为核心指标，定量评估模型在各类数据集与场景中的整体分割性能。
2. 创新精细化评估策略：提出基于高度的层级化评估方法，以植物点沿Z轴的最低高度为基准，以0.05m为步长划分6个垂直层，单独评估植物与非植物边界区域的分割精度，可实现对边界分割效果的精细化检验，尤其适配地形起伏、低矮植被、背景噪声的复杂场景。

## 3. 结果

### 3.1 大田植物群体的分割结果

核心内容：

1. 对比方案：与3类传统几何方法（Grid Ground、RANSAC、CSF）、3类主流深度学习模型（PointNet++、PointCNN、RandLANet）开展全面对比，所有传统方法均通过网格搜索确定最优参数，保证对比的公平性与性能最大化。
2. 整体性能结果：PlaneSegNet的整体平均IoU达到0.9903、平均F1分数达到0.9951，显著优于所有传统方法与主流深度学习模型；深度学习方法的整体性能远超传统几何方法，核心原因是传统方法难以处理密集玉米群体中植物与非植物的模糊边界。
3. 边界区域性能结果：在植物与非植物的边界区域，PlaneSegNet在各个垂直层的mIoU均大幅领先其他对比模型，最贴近边界的0.05m层mIoU达到0.7501，远超第二名RandLANet的0.5404，直接证明了平面注意力模块对边界分割精度的提升效果。
4. 推理效率结果：尽管PlaneSegNet的参数量（57.28M）远大于对比模型，但其单场景推理速度最快，仅需0.23s，结构化、高并行度的算子让模型同时兼具精度与效率，具备大田场景实际部署的条件。

### 3.2 温室环境中的分割结果

核心内容：

1. 实验目的：采用Crop3D温室数据集，进一步评估PlaneSegNet在不同场景中的可扩展性与泛化能力，应对温室环境中人工设施、光照多变、局部遮挡、结构反射等独特挑战。
2. 性能结果：PlaneSegNet在所有作物类型中均实现了最优性能，平均IoU达到0.9594、平均F1分数达到0.9787，远超PointNet++、PointCNN、RandLANet等对比模型。
3. 可视化验证结果：对比模型在盆架、地面反射等边界区域出现了明显的错分，而PlaneSegNet凭借平面注意力模块，有效捕捉了XZ与YZ平面的投影几何线索，大幅降低了植物-背景过渡区域的分割误差，输出了更稳定的预测结果，验证了模型在温室复杂环境、多物种场景中的优异泛化能力。

### 3.3 乡村机载激光雷达点云数据的分割结果

核心内容：

1. 实验目的：采用DALES数据集，在大规模、高噪声场景中评估PlaneSegNet的分割性能与鲁棒性。
2. 性能结果：PlaneSegNet的整体mIoU达到0.810，超过了RandLANet的0.793，在植被、轿车、卡车、围栏、建筑等复杂目标类型的分类中，均实现了比对比模型更高的精度。
3. 可视化验证结果：模型对纤细的垂直结构与宽阔的平面表面均实现了清晰的边界刻画，即使面对点云稀疏、地形复杂的挑战，仍保持了极低的错分率，证明模型在大规模、地形多样的点云数据中具备优异的泛化能力，可适配精准农业、生态监测、城市土地覆盖分类等多类场景。

### 3.4 消融实验

核心内容：

1. 实验目的：通过6组基于稀疏3D U-Net骨干的变体模型对比，系统验证平面注意力模块，以及不同平面投影聚合对分割精度（尤其是边界区域）的贡献。
2. 整体性能结果：各模型的整体IoU与F1分数差异极小，核心原因是植物与非植物的边界区域在整个点云中占比极低，全局指标无法体现边界区域的性能差异。
3. 边界区域性能结果：
   - 仅加入XY平面投影，不仅没有提升边界性能，反而出现了轻微下降，因为XY地平面视图易受背景杂波干扰，能提供的判别性信息极少。
   - 单加入XZ或YZ平面投影，模型在所有高度层的边界性能均有提升，且两个单平面变体的性能十分接近，证明这两个垂直平面对几何特征提取具备互补性。
   - 同时加入XY+XZ+YZ三平面的模型，性能低于XZ+YZ双平面设计，因为XY平面的信息会稀释垂直平面对植物/非植物过渡区的关键判别线索。
   - 融合XZ+YZ双平面的PlaneSegNet，在所有高度层均实现了最高精度，最贴近边界的0.05m、0.10m层mIoU显著领先其他所有变体，充分证明平面注意力模块通过双垂直平面特征融合，能大幅提升边界区域的分割效果。

## 4. 讨论

### 4.1 PlaneSegNet架构的优势

核心内容：

1. 对比传统几何方法：传统方法仅在平坦地形、稀疏植被区域表现较好，面对结构复杂的密集作物群体点云时，受地形起伏、杂草干扰、边界模糊的影响，分割精度会显著下降，而PlaneSegNet基于深度学习的特征提取能力，能很好地适配复杂农业场景。
2. 对比PointNet++、PointCNN等基于点的模型：这类模型依赖K近邻、球查询等基于邻域的特征聚合策略，而PlaneSegNet将原始点云转换为稀疏3D体素栅格，仅对非空体素执行卷积操作，大幅减少了冗余计算，提升了内存效率与推理速度，更适合处理包含数百万个点的大规模点云。
3. 对比RandLANet轻量化模型：RandLANet基于逐点聚合，在局部结构建模、空间连续性表征、细粒度边界刻画方面存在局限，在密集遮挡的农业场景中易出现预测模糊与边界错分；而PlaneSegNet通过体素建模保留了点云的空间拓扑结构，通过稀疏卷积构建了统一高效的3D感受野，避免了点采样与特征聚合过程中的信息损失，在多类场景中的精度、稳定性、边界保留效果均持续优于RandLANet。

### 4.2 平面注意力在边界感知分割中的有效性

核心内容：

1. 农业场景的核心痛点：植物与非植物元素的边界区域，往往存在高程相近、结构渐变、轮廓模糊的特点，缺乏清晰的几何可分性，易出现频繁错分与边界模糊，直接影响下游表型测量的精度。
2. 平面注意力的解决逻辑：通过将3D点云特征投影到XZ、YZ正交平面，增强了模型对空间结构变化的方向敏感性，有效捕捉了植物形态在垂直方向的细微变化，弥补了稀疏卷积固有的方向不敏感性，大幅提升了模型区分细粒度结构边界的能力。
3. 实验验证：消融实验结果证明，融合XZ、YZ双平面投影特征的PlaneSegNet，在边界区域的多个垂直层中，均持续取得了比仅用稀疏卷积、单平面聚合模型更高的mIoU，不仅提升了整体分割精度，还实现了关键区域的精细几何建模，尤其适合复杂农业环境中结构密集、易受遮挡的作物群体场景。

### 4.3 在DALES数据集上的几何感知泛化能力

核心内容：

1. 跨域泛化表现：尽管PlaneSegNet是为植物与非植物的二分类分割任务设计，但其在DALES多类别基准数据集上，相比稀疏3D U-Net基线模型，整体mIoU提升了2.9%，展现出了显著的跨域泛化能力。
2. 性能提升的规律：模型的性能提升具有强烈的类别依赖性，主要集中在卡车、电线杆、围栏、电线等依赖垂直截面几何特征进行区分的类别，在其他类别中也有幅度适中但稳定的提升。
3. 核心结论：平面注意力模块能放大细长轮廓、直立结构、丝状元素在XZ/YZ投影中的显著特征，锐化这类结构的决策边界；而这种提升植物/非植物界面刻画能力的归纳偏置，能有效迁移到语义由三维形状决定的非植物类别中。

### 4.4 稀疏地面场景下的性能表现

核心内容：

1. 实验场景与目的：针对作物生育后期冠层高度闭合、地面点稀疏不连续的严苛场景，通过无人机多视角影像重建了成熟期密集玉米点云，使用前期训练好的模型权重零微调直接测试，验证模型在极端场景下的泛化能力。
2. 性能结果：基于点的模型（PointNet++、PointCNN、RandLANet）基本无法恢复地面类别，地面IoU几乎为0；基于体素的稀疏3D U-Net表现出一定韧性，地面IoU达到0.6613；而PlaneSegNet实现了近乎完美的性能，地面IoU达到0.9774、植物IoU达到0.9984，大幅优于所有基线模型。
3. 性能提升原因：平面注意力模块聚合了XZ、YZ平面的投影特征，强化了植物-地面界面的垂直几何变化，即使地面特征碎片化，结合稀疏体素卷积保留的结构连续性，仍能鲁棒识别地面条带，清晰分离边界区域；这种特性对作物生育后期的无人机摄影测量任务至关重要，模型无需重新训练即可直接部署，适配地面稀疏的恶劣场景。

### 4.5 旋转扰动对分割性能的影响

核心内容：

1. 实验目的：验证模型在点云出现垂直方向旋转扰动时的鲁棒性，模拟真实农业场景中传感器倾斜、安装误差的实际情况；对大田玉米点云施加围绕X、Y轴±5°、±10°、±15°的旋转，评估地面-植物边界区域的分割性能。
2. 实验结果：施加额外的旋转扰动后，PlaneSegNet的边界区域分割精度仅出现小幅下降，且始终保持比稀疏3D U-Net基线更高的mIoU，同时持续优于PointNet++、RandLANet等模型。
3. 核心结论：PlaneSegNet对点云垂直方向的旋转偏差具备优异的容忍度，在真实农业采集场景中，即使无法保证地面法向量与采集坐标系Z轴完全对齐，仍能输出稳定可靠的分割结果，实用价值更强；未来在训练过程中融入更大角度的X、Y轴旋转，可进一步提升模型的鲁棒性。

### 4.6 在自动化植物表型测定中的应用潜力

核心内容：

1. 数据适配性：PlaneSegNet可兼容地面激光扫描、无人机摄影测量、机载激光雷达等多种传感模态生成的点云，在大田、温室等不同环境、不同空间尺度的数据集上，均能保持稳定的分割性能。
2. 物种适配性：模型不依赖特定植物物种、生育期的先验知识，已在玉米、油菜、马铃薯等多种作物的群体点云上验证了有效性，跨物种适应性极强。
3. 实用价值：可作为高通量表型工作流的核心预处理模块，高效剔除地面、支撑结构、土壤、杂草等冗余背景点，为下游表型分析提供干净、高精度的纯植物点云；大幅降低了大规模点云人工分割的成本与主观性，推动了群体植物感知任务的自动化进程，可无缝集成到现有的农业遥感平台与三维表型系统中，可部署性与可扩展性极强。

### 4.7 局限性

核心内容：

1. 标注依赖局限：模型基于监督学习构建，需要高质量的标注点云，但目前群体级农业数据集仍缺乏标准化的标注协议，复杂农业环境中人工标注的成本高、主观性强，限制了模型的大规模部署与泛化能力。
2. 超大规模数据处理局限：尽管模型采用稀疏体素卷积提升了效率，但在处理城市级机载激光雷达、超大面积农田无人机重建点云等超大规模数据时，仍可能面临内存与推理速度的限制；分块推理的解决方案会引入边界不连续、拼接伪影的问题，未来需要聚焦高效的场景划分方法、块间特征一致性保持策略，以及轻量化网络架构，进一步提升模型的可扩展性与部署可行性。

## 5. 结论

核心内容：

1. 本文提出了专为复杂作物群体场景植物点云高效、精准提取设计的PlaneSegNet语义分割网络，针对植物与非植物结构间几何过渡模糊、边界不清晰的行业核心痛点，将基于体素的稀疏卷积架构与全新的平面注意力模块相融合，通过聚合XZ与YZ垂直平面的投影特征，增强了模型对垂直结构变化的敏感性，大幅提升了边界区域的分割精度。
2. 多场景全面验证：在不同生育期、种植密度的大田玉米群体点云，背景结构复杂的温室盆栽作物点云，以及DALES大规模机载激光雷达数据这三类代表性场景中，PlaneSegNet在整体分割精度、边界区域精度、跨场景泛化能力方面，均持续优于传统几何方法与主流深度学习基线模型。
3. 应用价值：通过稀疏三维卷积与方向特征增强的结合，PlaneSegNet为三维点云中的自动化植物分割提供了一套实用、可扩展的解决方案，高度适配株高估算、冠层建模、单株刻画等下游植物表型任务，其在不同数据源、不同作物类型中稳定的性能表现，凸显了其作为精准农业高通量表型工作流预处理模块的巨大应用潜力。

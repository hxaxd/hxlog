# 书籍评估

## 目标与评估方法

- 目标是理解计算机科学的大部分核心问题, 并在人工智能, Agent 与 AI 系统方向形成纵深
- 书籍负责提供稳定的概念结构, 课程负责组织学习, 官方文档负责更新实现, 论文负责进入研究前沿
- 全面的候选列表不等于全部精读, 每个领域应选一条主线, 其余作为补充或参考
- 阅读优先级分为核心, 高, 中, 低
- 建议结论分为主线, 补充, 参考, 暂不采用
- 审批状态由本人逐本改为接受, 备选或拒绝, 初始状态统一为待审批

## 人工智能与机器学习

### Artificial Intelligence: A Modern Approach

- 版本: 4th Edition, 2020
- 领域: 人工智能全貌
- 定位: 用一本书建立人工智能作为完整学科的概念地图
- 主要内容: 智能体, 搜索, 约束满足, 博弈, 逻辑, 知识表示, 概率推理, 决策, 机器学习, NLP, 机器人与 AI 安全
- 与现有资料的重叠: 与离散数学, 计算理论, 统计学习方法和 Agent 总览局部重叠, 但现有资料没有统一这些主题
- 优势: 覆盖面极广, 能纠正人工智能等同于深度学习或大模型的偏差
- 局限: 体量大, 各专题深度有限, 深度学习和 LLM 部分不是当前前沿教程
- 阅读优先级: 核心
- 建议结论: 作为人工智能全貌主线, 选择性精读
- 审批状态: 待审批
- 资料: [Pearson](https://www.pearson.com/en-us/subject-catalog/p/artificial-intelligence-a-modern-approach/P200000003500)

### Probabilistic Machine Learning

- 版本: An Introduction, 2022; Advanced Topics, 2023
- 领域: 概率机器学习
- 定位: 用概率建模统一经典机器学习, 深度学习与现代生成模型
- 主要内容: 概率论, 统计决策, 线性模型, 核方法, 图模型, 近似推断, 深度生成模型, 高斯过程, 贝叶斯方法与强化学习
- 与现有资料的重叠: 覆盖并显著扩展数学基础中的概率统计和现有的统计学习方法
- 优势: 现代, 数学统一性强, 可作为长期查询手册
- 局限: 数学密度高, 两卷规模过大, 不适合第一次接触机器学习时线性通读
- 阅读优先级: 高
- 建议结论: 第一卷作为理论主线或参考, 第二卷按研究主题选读
- 审批状态: 待审批
- 资料: [Probabilistic Machine Learning](https://probml.github.io/pml-book/)

### Understanding Deep Learning

- 版本: 2023
- 领域: 深度学习
- 定位: 在可读性和现代性之间取得平衡的深度学习原理主线
- 主要内容: 监督学习, 深度网络, 优化, 正则化, CNN, Transformer, 图神经网络, 生成模型与强化学习
- 与现有资料的重叠: 与李宏毅课程和卷积神经网络笔记重叠, 但覆盖更系统且延伸到 Transformer 和生成模型
- 优势: 图示清楚, 数学推导适中, 内容比 2016 年的 Deep Learning 更接近当前模型
- 局限: 工程实现和大规模训练系统不是重点
- 阅读优先级: 核心
- 建议结论: 作为现代深度学习第一主线
- 审批状态: 待审批
- 资料: [Understanding Deep Learning](https://udlbook.github.io/udlbook/)

### Dive into Deep Learning

- 版本: 在线持续更新版, 纸质版 2023
- 领域: 深度学习实践
- 定位: 将数学, 模型和可运行代码放在一起的实践型教材
- 主要内容: 线性模型, MLP, CNN, RNN, Attention, Transformer, 优化, 计算性能, 计算机视觉和 NLP
- 与现有资料的重叠: 与深度学习, 卷积神经网络和 Python 数据处理框架高度重叠
- 优势: 免费, 可执行, 适合边学边验证, 能补足只看理论缺少实现的问题
- 局限: 理论统一性弱于 Probabilistic Machine Learning, 代码框架可能随版本变化
- 阅读优先级: 高
- 建议结论: 作为 Understanding Deep Learning 的配套实践, 不单独承担理论主线
- 审批状态: 待审批
- 资料: [Dive into Deep Learning](https://d2l.ai/)

### Deep Learning

- 版本: 2016
- 领域: 深度学习理论
- 定位: 深度学习早期成熟阶段的经典理论参考
- 主要内容: 线性代数, 概率, 数值计算, 前馈网络, 正则化, 优化, CNN, RNN, 表示学习与生成模型
- 与现有资料的重叠: 与数学基础, Understanding Deep Learning 和现有深度学习笔记大量重叠
- 优势: 基础章节严谨, 对优化, 正则化和表示学习的讨论仍有参考价值
- 局限: 早于 Transformer, Diffusion 和现代大模型, 不再适合作为唯一主线
- 阅读优先级: 中
- 建议结论: 作为经典参考书按主题查阅, 不建议从头通读
- 审批状态: 待审批
- 资料: [Deep Learning](https://www.deeplearningbook.org/)

### An Introduction to Statistical Learning with Applications in Python

- 版本: 2023
- 领域: 统计学习
- 定位: 面向实践的经典机器学习入门
- 主要内容: 回归, 分类, 重采样, 正则化, 树模型, SVM, 无监督学习, 深度学习和生存分析
- 与现有资料的重叠: 与统计学习方法和机器学习笔记高度重叠, 但更强调数据分析与 Python 实验
- 优势: 可读性强, 实例完整, 适合补足从统计概念到实际建模的路径
- 局限: 数学深度有限, 对现代生成模型和大模型覆盖较少
- 阅读优先级: 中
- 建议结论: 若统计学习方法读起来过于抽象则采用, 否则作为实践补充
- 审批状态: 待审批
- 资料: [ISLP](https://www.statlearning.com/)

### Speech and Language Processing

- 版本: 3rd Edition 在线草稿, 持续更新
- 领域: 自然语言处理与语言模型
- 定位: 从传统 NLP 到 Transformer 和大语言模型的完整参考
- 主要内容: 文本处理, 分类, Embedding, 序列模型, Transformer, 预训练语言模型, 信息抽取, 问答, 机器翻译, 语音与对话系统
- 与现有资料的重叠: 与 Agent, RAG 和深度学习局部重叠, 但补足语言模型及 NLP 的学科基础
- 优势: 覆盖完整, 免费更新, 适合解释 Agent 下层模型能力来自哪里
- 局限: 篇幅很大, 工具调用和 Agent 工程不是重点, 在线草稿结构会变化
- 阅读优先级: 高
- 建议结论: NLP 与 LLM 方向的长期参考, 按主题阅读
- 审批状态: 待审批
- 资料: [Speech and Language Processing](https://web.stanford.edu/~jurafsky/slp3/)

### Reinforcement Learning: An Introduction

- 版本: 2nd Edition, 2018
- 领域: 强化学习
- 定位: 强化学习的标准概念与算法主线
- 主要内容: 多臂老虎机, MDP, 动态规划, Monte Carlo, 时序差分, 函数逼近, 策略梯度与规划
- 与现有资料的重叠: 与 Agent 的规划和反馈概念只有表面交集, 现有目录缺少强化学习理论
- 优势: 概念严谨, 算法发展脉络清晰, 免费
- 局限: 深度强化学习和 RLHF 不是全书重点, 需要概率与优化基础
- 阅读优先级: 中
- 建议结论: 保留为 AI 全貌的重要专题, 在需要 RL 或 RLHF 时提高优先级
- 审批状态: 待审批
- 资料: [Reinforcement Learning: An Introduction](http://incompleteideas.net/book/the-book-2nd.html)

### Build a Large Language Model From Scratch

- 版本: 1st Edition, 2024
- 领域: 大语言模型
- 定位: 通过实现 GPT 类模型理解训练与微调流程
- 主要内容: Tokenization, Attention, Transformer, 预训练数据管线, 训练循环, 分类微调和指令微调
- 与现有资料的重叠: 与深度学习和未来 AI 系统笔记重叠, 但现有内容缺少端到端语言模型实现
- 优势: 目标明确, 代码可执行, 能把模型结构与训练过程连接起来
- 局限: 规模较小, 不能替代分布式训练, 推理系统和现代后训练资料
- 阅读优先级: 高
- 建议结论: 作为进入 LLM 内部机制的实践主线
- 审批状态: 待审批
- 资料: [Manning](https://www.manning.com/books/build-a-large-language-model-from-scratch)

## 数据工程与大数据

### Fundamentals of Data Engineering

- 版本: 1st Edition, 2022
- 领域: 数据工程
- 定位: 建立数据工程生命周期和架构决策的全貌
- 主要内容: 数据生成, 摄取, 存储, 转换, 服务, 编排, 安全, DataOps, 数据质量和架构选择
- 与现有资料的重叠: 与数据库, 分布式, 容器和系统设计局部重叠, 但现有资料缺少完整数据生命周期
- 优势: 领域边界清楚, 技术中立, 适合建立数据工程总览
- 局限: 算法和具体引擎实现较浅, 部分云产品案例会随时间变化
- 阅读优先级: 核心
- 建议结论: 作为数据工程全貌主线
- 审批状态: 待审批
- 资料: [O'Reilly](https://www.oreilly.com/library/view/fundamentals-of-data/9781098108298/)

### Designing Data-Intensive Applications

- 版本: 2nd Edition, 2026
- 领域: 数据系统与分布式系统
- 定位: 理解可靠, 可扩展, 可维护数据系统的核心设计权衡
- 主要内容: 数据模型, 存储与检索, 编码, 复制, 分区, 事务, 分布式系统, 批处理与流处理
- 与现有资料的重叠: 与分布式系统, 数据库系统概念和系统设计重叠, 但更强调跨系统的设计原则
- 优势: 把数据库, 分布式和数据处理连接成统一知识网络, 与目标高度一致
- 局限: 不是数据库实现教材, 也不是具体大数据框架操作手册
- 阅读优先级: 核心
- 建议结论: 接受为数据系统主线, 优先完成现有空白笔记
- 审批状态: 待审批
- 资料: [O'Reilly](https://www.oreilly.com/library/view/designing-data-intensive-applications/9781098119058/)

### Mining of Massive Datasets

- 版本: 3rd Edition, 2020
- 领域: 大数据算法
- 定位: 学习大规模数据上的算法思想而不是某个具体框架
- 主要内容: MapReduce, 相似项发现, 数据流, 链接分析, 频繁项集, 聚类, 推荐系统, 图挖掘和降维
- 与现有资料的重叠: 与算法, 机器学习和分布式计算局部重叠, 但补充规模化算法视角
- 优势: 免费, 经典, 将算法复杂度与分布式执行结合
- 局限: 部分系统背景偏 MapReduce 时代, 不覆盖完整数据工程生命周期
- 阅读优先级: 高
- 建议结论: 作为大数据算法主线, 选择性精读
- 审批状态: 待审批
- 资料: [Mining of Massive Datasets](http://www.mmds.org/)

### Streaming Systems

- 版本: 1st Edition, 2018
- 领域: 流处理
- 定位: 深入理解无界数据处理的时间与一致性语义
- 主要内容: 事件时间, 处理时间, 窗口, Watermark, Trigger, 状态, Exactly-once 和批流统一
- 与现有资料的重叠: 与 DDIA 的流处理部分重叠, 但在时间语义和流系统模型上更深入
- 优势: 解释流处理最容易混淆的核心概念, 不只是某个 API 教程
- 局限: Google Beam 语境较强, 不适合作为数据工程第一本书
- 阅读优先级: 中
- 建议结论: 完成数据工程和 DDIA 后按需补充
- 审批状态: 待审批
- 资料: [O'Reilly](https://www.oreilly.com/library/view/streaming-systems/9781491983867/)

### The Data Warehouse Toolkit

- 版本: 3rd Edition, 2013
- 领域: 数据仓库与维度建模
- 定位: 学习事实表, 维度表和分析型数据建模
- 主要内容: 星型模式, 维度建模技术, 缓慢变化维度以及行业案例
- 与现有资料的重叠: 与数据库设计和数据工程局部重叠, 现有内容没有分析型建模体系
- 优势: 维度建模的经典参考, 实际数据仓库场景覆盖广
- 局限: 年代较早, 对数据湖, 湖仓和现代流式架构覆盖不足
- 阅读优先级: 中
- 建议结论: 作为数据仓库专题参考, 不承担数据工程总览
- 审批状态: 待审批
- 资料: [Wiley](https://www.wiley.com/en-us/The+Data+Warehouse+Toolkit%3A+The+Definitive+Guide+to+Dimensional+Modeling%2C+3rd+Edition-p-9781118530801)

### Data Mining: Concepts and Techniques

- 版本: 4th Edition, 2022
- 领域: 数据挖掘
- 定位: 系统介绍从数据预处理到模式发现的传统数据挖掘方法
- 主要内容: 数据清洗, 数据仓库, 关联规则, 分类, 聚类, 异常检测, 图与复杂数据挖掘
- 与现有资料的重叠: 与机器学习, MMDS 和数据仓库内容重叠较大
- 优势: 覆盖传统数据挖掘全貌, 适合作为术语和方法参考
- 局限: 若已学习现代机器学习和 MMDS, 线性通读的边际收益有限
- 阅读优先级: 低
- 建议结论: 作为参考书保留, 暂不列为主线
- 审批状态: 待审批
- 资料: [Elsevier](https://www.sciencedirect.com/book/9780128117606/data-mining-concepts-and-techniques)

## 并行计算与 AI 系统

### An Introduction to Parallel Programming

- 版本: 2nd Edition, 2021
- 领域: 通用并行计算
- 定位: 从共享内存, 消息传递和 GPU 三种模型建立并行编程主线
- 主要内容: 并行硬件, 性能分析, MPI, Pthreads, OpenMP, CUDA 和并行程序设计
- 与现有资料的重叠: 与计算机组成中的并行处理器, OSTEP 并发和 C++ 并发局部重叠
- 优势: 模型完整, 能连接硬件知识与实际并行程序
- 局限: AI 工作负载不是重点, API 细节需要配合最新官方文档
- 阅读优先级: 高
- 建议结论: 作为通用并行计算主线
- 审批状态: 待审批
- 资料: [Elsevier](https://shop.elsevier.com/books/an-introduction-to-parallel-programming/pacheco/978-0-12-804605-0)

### Programming Massively Parallel Processors

- 版本: 5th Edition, 2026
- 领域: GPU 与 CUDA
- 定位: 从 GPU 架构, CUDA 编程到高性能并行算法的核心教材
- 主要内容: CUDA 模型, 内存层次, 性能优化, 卷积, 归约, 扫描, 稀疏计算, 图算法和深度学习工作负载
- 与现有资料的重叠: 与计算机组成的 GPU 章节, 卷积神经网络和 DCNv3 算子开发经验重叠, 但体系更完整
- 优势: 与 AI Infra 目标直接相关, 将硬件模型和 Kernel 优化结合
- 局限: 聚焦 NVIDIA CUDA, 不能替代通用并行理论或分布式训练
- 阅读优先级: 核心
- 建议结论: 作为 GPU 计算主线
- 审批状态: 待审批
- 资料: [Elsevier](https://shop.elsevier.com/books/programming-massively-parallel-processors/hwu/978-0-443-43900-1)

### Structured Parallel Programming

- 版本: 1st Edition, 2012
- 领域: 并行设计模式
- 定位: 使用可复用模式组织并行算法与程序结构
- 主要内容: Map, Reduce, Scan, Pipeline, Divide and Conquer, Task Graph 和并行模式组合
- 与现有资料的重叠: 与算法设计, C++ 并发和通用并行编程重叠
- 优势: 强调问题结构和模式, 有助于跨语言迁移
- 局限: 若已采用完整并行教材, 单独通读的必要性不高
- 阅读优先级: 中
- 建议结论: 作为并行设计补充, 不作为第一主线
- 审批状态: 待审批
- 资料: [Elsevier](https://shop.elsevier.com/books/structured-parallel-programming/mccool/978-0-12-415993-8)

### Efficient Processing of Deep Neural Networks

- 版本: 2020
- 领域: DNN 计算架构
- 定位: 理解神经网络计算, 数据流, 加速器和软硬件协同优化

- 优势: 能连接模型结构, 算子成本和硬件架构
- 局限: 早于当前 LLM 推理系统浪潮, 软件系统和分布式训练覆盖有限
- 阅读优先级: 中
- 建议结论: 作为 AI 芯片与算子优化专题参考
- 审批状态: 待审批
- 资料: [Efficient Processing of Deep Neural Networks](https://www.morganclaypoolpublishers.com/catalog_Orig/product_info.php?products_id=1606)

- 优势: 更贴近当前 AI 系统问题, 可作为专题入口
- 局限: 在线内容仍在演进, 不适合替代稳定的基础教材
- 阅读优先级: 高
- 建议结论: 在 Volume 1 之后按 Agent 与基础模型需求选读
- 审批状态: 待审批
- 资料: [Machine Learning Systems](https://www.mlsysbook.ai/)

### Designing Machine Learning Systems

- 版本: 1st Edition, 2022
- 领域: 生产级机器学习系统
- 定位: 从工业实践角度理解数据, 模型, 部署和监控之间的设计权衡
- 主要内容: 需求定义, 数据分布, 特征工程, 模型开发, 部署, 数据漂移, 监控和持续学习
- 与现有资料的重叠: 与 Machine Learning Systems Volume 1 和数据工程高度重叠
- 优势: 实践决策密度高, 对真实系统失败模式和组织流程讨论充分
- 局限: 底层算子, GPU 和分布式训练不是重点
- 阅读优先级: 高
- 建议结论: 作为 ML Systems 主线的工业实践补充
- 审批状态: 待审批
- 资料: [O'Reilly](https://www.oreilly.com/library/view/designing-machine-learning/9781098107956/)

### AI Engineering

- 版本: 1st Edition, 2025
- 领域: 基础模型应用系统
- 定位: 面向使用基础模型构建产品与系统的工程方法
- 主要内容: 模型选择, 评测, Prompt, RAG, Agent, 微调, 数据, 推理优化, 监控与反馈
- 与现有资料的重叠: 与现有 Agent, RAG, 评测和 AI 工具笔记高度重叠
- 优势: 正好连接基础模型能力与 Agent 工程, 能为现有零散笔记提供统一结构
- 局限: 技术变化快, 不能替代模型原理或底层 AI Infra 资料
- 阅读优先级: 核心
- 建议结论: 作为当前 Agent 工作方向的工程主线
- 审批状态: 待审批
- 资料: [O'Reilly](https://www.oreilly.com/library/view/ai-engineering/9781098166298/)

## 工具栏整体评估

- 当前覆盖: 文档格式, Git, Linux, Shell, 终端, 序列化, Nginx, Web, 容器和前端基础已经足够广
- 主要问题: 工具, 系统管理, 部署组件和 Web 常识处于同一级, 但本轮不调整结构
- 适合书籍的内容: Git 原理, Linux 整体机制, Shell 基础和系统管理等相对稳定知识
- 适合官方文档的内容: Docker, Kubernetes, Nginx, FastAPI, 前端框架, 编辑器, 终端工具和具体云服务
- 待补缺口: Python 环境, 依赖与项目管理
- 待补缺口: 调试, 性能分析, 日志, Metrics, Tracing 与可观测性
- 待补缺口: SSH, 远程开发, 集群管理和 Slurm 等任务调度
- 待补缺口: 实验追踪, 数据版本, 模型版本和制品管理

    Agent = зӢ¬з«ӢдёҠдёӢж–Үзҡ„иў«иҝһз»ӯи°ғз”Ёзҡ„ LLM + ж”Ҝж’‘е…¶е·ҘдҪңзҡ„ Harness жЎҶжһ¶ + е·ҘдҪңзҺҜеўғ

    ж ёеҝғ

    • Agent йўҶеҹҹзҡ„е·ҘдҪңжҳҜеҸ‘жҺҳжЁЎеһӢдёҺзҺҜеўғзҡ„жӣҙй«ҳж•Ҳзҡ„дәӨдә’ж–№ејҸ
    • й«ҳж•ҲжҖ§зҡ„жқҘжәҗзӣҙжҺҘеҸ–еҶідәҺжЁЎеһӢзҡ„и®ӯз»ғж•°жҚ®
    • ж—©жңҹе·ҘдҪңжң¬иҙЁдёҠжҳҜеңЁеҸ‘жҺҳиҮӘз„¶иҜӯиЁҖж•°жҚ®дёӯи•ҙеҗ«зҡ„дәӨдә’жЁЎејҸ, з§°д№Ӣдёә Prompt Engineering
    • йҡҸзқҖжӣҙеӨҡеҗҺеӨ©дәӨдә’жЁЎејҸеҸҚдҪңз”ЁдәҺж•°жҚ®зҡ„еҪұе“Қ + жЁЎеһӢиғҪеҠӣзҡ„жҸҗеҚҮ, дәӨдә’жЁЎејҸе·ІдёҚжҳҜз“¶йўҲ
      • жЁЎеһӢжҳҜеҗҰжӢҘжңүи¶іеӨҹзҡ„еҜ№зҺҜеўғзҡ„еҜ№дәӨдә’ж–№ејҸзҡ„зҹҘиҜҶдёәдё»иҰҒдјҳеҢ–ж–№еҗ‘, з§°д№Ӣдёә Context Engineering
    • зӣҙиҮід»ҠеӨ©, й•ҝзЁӢд»»еҠЎзҡ„ж•°жҚ®иһҚдәҺжЁЎеһӢеҗҺи®ӯз»ғиҝҮзЁӢ, жЁЎеһӢеҖҫеҗ‘еҸҳжҲҗй«ҳеәҰеҸҜе®ҡеҲ¶зҡ„иғҪеҠӣ, еӣҙз»•иҜҘиғҪеҠӣйӣҶзҡ„е®Ңж•ҙи®ҫи®Ўж ҮеҮҶ, з§°д№Ӣдёә Harness Engineering

    жҰӮеҝө

    • Harness жҳҜжЁЎеһӢдёҺзҺҜеўғдәӨдә’зҡ„дёӯд»Ӣ, е»әз«ӢеңЁзҺҜеўғзҡ„еҸҜи§ЈжһҗжҖ§дёҺжЁЎеһӢзҡ„е…ҲйӘҢиғҪеҠӣд№ӢдёҠ
    • Harness иҙҹиҙЈз»ҙжҠӨиӢҘе№ІдёӘжЁЎеһӢзҠ¶жҖҒ (Context), д№ҹиҙҹиҙЈдёҺзҺҜеўғзҡ„жҺҘеҸЈ (Runtime), д»ҘеҸҠ Harness иҮӘиә«зҡ„иҝӣеҢ– (Evolution)

    Context

    • Knowledge жҳҜ Context дёӯе…ідәҺзҺҜеўғе’Ңд»»еҠЎзҡ„дҝЎжҒҜ
      • Knowledge иЎЁзӨәжҸҗдҫӣз»ҷжЁЎеһӢзҡ„дҝЎжҒҜ, дёҚд»ЈиЎЁе…¶дёҖе®ҡе®ўи§Ӯдёәзңҹ, е…¶дёӯеҸҜд»ҘеҢ…жӢ¬дәӢе®һ, и§ӮеҜҹ, еҒҮи®ҫ, йў„жөӢе’Ңзӣёдә’еҶІзӘҒзҡ„дҝЎжҒҜ
      • Reasoning жҳҜжЁЎеһӢеҜ№ Knowledge зҡ„еҸҳжҚў, д»Ҙдә§з”ҹеҲӨж–ӯ, йў„жөӢ, и§ЈйҮҠжҲ–иЎҢеҠЁдҫқжҚ®
        • Reasoning йҖҡеёёйҷҚдҪҺдёҺ Objective зӣёе…ізҡ„дёҚзЎ®е®ҡжҖ§
        • жҺўзҙўйҳ¶ж®өд№ҹеҸҜиғҪйҖҡиҝҮз”ҹжҲҗеҒҮи®ҫдё»еҠЁеўһеҠ еҖҷйҖүи§ЈйҮҠ
      • дёҠдёӢж–ҮеҺӢзј©йҖҡиҝҮеҲ йҷӨеҶ—дҪҷзҡ„ Knowledge жқҘжҸҗй«ҳжңүж•ҲдҝЎеҸ·еҜҶеәҰ
    • Objective жҳҜдёҖз§Қзү№ж®Ҡзҡ„ Knowledge, жҸҸиҝ°еҜ№зҺҜеўғзҠ¶жҖҒ, иЎҢеҠЁиҪЁиҝ№жҲ–жңҖз»Ҳз»“жһңзҡ„еҒҸеҘҪ
      • еҢ…жӢ¬еёҢжңӣиҫҫеҲ°зҡ„ Goal, дёҚеҸҜиҝқеҸҚзҡ„ Constraint е’ҢеҲӨж–ӯе®ҢжҲҗзҡ„ Completion Criteria
      • жЁЎеһӢеҖҫеҗ‘дәҺжңҚд»Һ Objective, е№¶еӣҙз»• Objective иҝӣиЎҢжҺЁзҗҶдёҺи§„еҲ’
      • Planning жҳҜж №жҚ® Objective, еҪ“еүҚ Belief дёҺ Action зҡ„еҸҜиғҪеҗҺжһңз»„з»ҮжңӘжқҘиЎҢеҠЁ
    • Cognitive Control жҳҜ Context дёӯе…ідәҺеҰӮдҪ•еӨ„зҗҶ Knowledge зҡ„жЁЎејҸ
      • зӣҙжҺҘеҪұе“ҚжЁЎеһӢзҡ„е…іжіЁзӮ№, жҠҪиұЎеұӮзә§, жҺЁзҗҶж–№еҗ‘е’ҢиЎҢеҠЁеҖҫеҗ‘
      • еҸҜд»ҘзҗҶи§ЈдёәжҝҖжҙ»жЁЎеһӢе·ІжңүиЎҢдёәжЁЎејҸзҡ„ index + еј•еҜјжЁЎеһӢзҡ„жіЁж„ҸеҲҶй…ҚдёҺз”ҹжҲҗеҖҫеҗ‘
      • дё»иҰҒеҪўејҸжҳҜжҸҗзӨәиҜҚжіЁе…Ҙ
      • Workflow жҳҜдёҖз§ҚжіЁе…Ҙ Cognitive Control зҡ„жңәеҲ¶, еҶіе®ҡеҪ“еүҚдҪҝз”Ёе“Әз§Қ Cognitive Control
    • Action жҳҜжЁЎеһӢиҫ“еҮәдёӯеҸҜиў« Runtime и§ЈйҮҠдёәзҺҜеўғдәӨдә’зҡ„йғЁеҲҶ
      • Action зҡ„зұ»еһӢ, еҸӮж•°еҸҠеҸҜиғҪеҗҺжһңдјҡдҪңдёә Knowledge жҸҗдҫӣз»ҷжЁЎеһӢ
      • жЁЎеһӢеҖҫеҗ‘дәҺе°Ҷжёҙжңӣеҫ—еҲ°зҡ„зҹҘиҜҶжҲ–зҺҜеўғзҠ¶жҖҒиҪ¬жҚўдёә Action, д»Ҙж»Ўи¶і Objective зҡ„иҰҒжұӮ
      • зҺҜеўғзҠ¶жҖҒеҜ№дәҺжЁЎеһӢжқҘиҜҙдёҚеҸҜзӣҙжҺҘи§ӮеҜҹ, еҸӘиғҪз”ұ Runtime иҪ¬жҚўдёә Observation еҗҺиҝӣе…Ҙ Context
    • Observation жҳҜ Runtime д»ҺзҺҜеўғдёӯеҸ–еҫ—е№¶жҸҗдҫӣз»ҷжЁЎеһӢзҡ„ Knowledge
      • Observation жҸҸиҝ°зҺҜеўғдёӯеҸ‘з”ҹдәҶд»Җд№Ҳ, дҪҶдёҚзӣҙжҺҘиЎЁзӨәз»“жһңзӣёеҜ№дәҺ Objective жҳҜеҗҰжӯЈзЎ®
      • жЁЎеһӢеҜ№ Observation зҡ„и§ЈйҮҠеұһдәҺ Reasoning, дёҚзӯүдәҺ Observation жң¬иә«
    • Feedback жҳҜ Evaluation дә§з”ҹе№¶жҸҗдҫӣз»ҷжЁЎеһӢзҡ„ Knowledge, з”ЁдәҺдҝ®жӯЈеҗҺз»ӯжҺЁзҗҶдёҺиЎҢеҠЁ
      • Feedback еҸҜд»ҘжқҘиҮӘзҺҜеўғ, жөӢиҜ•, дәәзұ», Critic жҲ–жЁЎеһӢиҮӘиә«, дёҚдёҖе®ҡз”ұ Action зӣҙжҺҘиҝ”еӣһ
      • жЁЎеһӢеҖҫеҗ‘дәҺй«ҳдј° Observation дёҺ Feedback зҡ„еҸҜдҝЎеәҰ, еӣ жӯӨйңҖиҰҒдҝқз•ҷе…¶жқҘжәҗдёҺеҸҜйӘҢиҜҒжҖ§
